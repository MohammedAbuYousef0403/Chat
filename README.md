# Chat App

#### Simple Chat created with React Native

## Technologies
This project created by:
* React Native
* Node.js (for server)
* MongoDB (for database)

.env in server
```
PORT       = 5000
DB_URL     = mongodb://localhost:27017/chat
JWT_SECRET = jwt-secret
```

.env in web
```
NODE_PATH        = src/
REACT_APP_SOCKET = ws://localhost:3001
```

.env in mobile
```
config/urls.js
```
